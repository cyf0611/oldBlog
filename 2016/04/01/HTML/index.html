SEO优化的设置：

  • <meta name="keywords" content="关键词，关键词">
  • <meta name="description" content="描述词，描述词">

<meta>标签提供关于 HTML 文档的元数据，元数据不会显示在页面上，但是对于机器是可读的。
典型的情况是，meta 元素被用于规定页面的描述、关键词、文档的作者、最后修改时间以及其他元数据。
<meta>标签始终位于 head 元素中。
元数据可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。

  • <base target="_blank">可以让当前页面中的所有的a标签都拥有相同的属性，比如target属性。【写在head标签中】

    • target属性：
    • 1、_self(默认的，表示在当前页面中打开超级链接）
    • 2、_blank 表示在新的页签中，或者是新的窗口中打开超级链接的页面。
    • 3、_top表示跳转到目标网址的顶部

  • <link rel="icon" href="图标地址(favicon.ico)">（设置网页标题图标）【网页小图标会放到根目录下、一般网页小图标后缀为.ico】

  • 在HTML文档中，文字之间的空格会进行合并，多个空格或者是换行会合并成一个空格。
    • 【&nbsp;可以表示一个空格】
    • 【&ensp;表示半个空格】
  • <i></i>定义斜体字（不推荐使用）
  • <em></em>标签是局部的文本的加强语气，或者是文本的强调的作用。可以会改变局部的文档的语义。
  • <strong></strong>标签是 针对整个文档而言，进行一个加强语气的作用。例如：一篇文章中最重要的主题思想或者段落用strong标签进行包裹
  • <ins></ins>文字以加下滑线的方式显示

  • 锚点链接（锚链接）：可以定义href属性指向一个页面中的id值，可以让页面跳转到id对应的标签的位置。

  • list-style: none;（去掉列表前圆点）

  • 快捷键： ctrl+{ 进行左缩进， ctrl+} 进行右缩进

【border=””显示表格边框粗细，值越大越粗，0表示没有边框】
【cellpadding=””单元格边距(Cell padding)增加单元格内容与其边框之间的距离、会影响表格大小】
【cellspacing=”(默认为2)”单元格间距(Cell spacing)增加单元格之间的距离】
【frame=””控制围绕表格的边框（box全部）（above上部）（below下部）（hsides上中下）（vsides左中右）】

合并单元格：
<td rowspan="2"（纵向）一个单元格占用两行（自己和它下面一行）进行显示>单元格内容</td><td>单元格内容</td><td>单元格内容</td>
<td colspan="2"(横向)一行中的一个单元格占用两个列（自己和它右面一行）进行显示>单元格内容</td><td>单元格内容</td><td>单元格内容</td>

form标签：表单的域标签，用于包裹整个表单的内容（获取用户数据）。表单就是用户提交数据到后台的一个虚拟单子。表单一般由文本框、下拉列表、单选、多选、文本域等组成。
action属性，用于指定当前表单 提交的时候指向后台的地址。
input标签:是文本框、单选、多选、按钮等。type属性不同的取值决定了input标签的作用。value属性是文本框的值。checked属性，值只有一个checked，可以省略属性值，在单选按钮和多选按钮中表示此按钮被选中

多选：checkbox

method=”（get|post）”（表单提交方式）【get将提交数据显示在地址栏中，不安全、post通过后台提交数据，安全】

label:
性别：男<input type="radio" name="gender" checked id="gender-man">【checked有此属性时表示默认被选中】
<label for="gender-man">男</label>
【label：当点击“男”字时也可选中选项；for：指向要提供服务的标签的id值，使其获得焦点】
<label><input type="radio" name="gender">女</label>
【lable两种用法：1、使用id；2、包裹】

select标签:用于下拉列表，或者列表标签，用option标签来设置选项，value值只有在后台有用。option包裹的内容是显示的选项的文本

multiple:是一个单属性值。值只有一个multiple设置了此属性表示select标签表现为列表标签,否则表示下拉列表标签,可多选
